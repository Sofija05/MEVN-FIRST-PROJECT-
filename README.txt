Treba ući u apis i app zatim i u jednom i u drugom treba iskoristiti yarn komandu u terminalu.
To ce instalirati sve potrebne pakete. 
Napraviti file development.env i upisati PORT = 4000
Prvo ući u folder apis iz terminala i kucati komandu yarn start, zatim u novom terminalu preći u 
folder app i isto kucati komandu yarn start
I kucati u browser-u localhost link koji izađe u terminalu.
